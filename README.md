# 💹 Forex Price Prediction

A machine learning project that predicts forex price direction using historical data.

---

## 🚀 Features
- Predicts price movement (up/down)  
- Uses a logistic regression model  
- Basic feature engineering with daily returns  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- Scikit-learn
